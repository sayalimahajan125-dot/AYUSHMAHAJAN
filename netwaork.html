Network Concepts

1. IP Address

Every device connected to a network has a unique IP (Internet Protocol) address ‚Äî like a digital home address. It ensures data reaches the right destination.

  • IPv4: 32-bit (e.g., 192.168.1.1)
  • IPv6: 128-bit (e.g., 2001:db8::1)

2. DNS (Domain Name System)

DNS converts human-friendly website names (like www.google.com) into machine-readable IP addresses.

3. Protocols

Protocols are standardized rules for communication between network devices:

  • HTTP/HTTPS: Web browsing
  • FTP: File transfers
  • SMTP/POP3: Email communication
  • TCP/IP: Foundation of the Internet

4. Router & Switch

Router: Directs data between networks.
Switch: Connects multiple devices in one local network.

5. Bandwidth

Bandwidth is the amount of data that can pass through a network per second ‚Äî higher bandwidth = faster internet.

6. Firewall

A firewall filters and monitors traffic to protect networks from unauthorized access or cyber threats.

7. LAN / WAN / MAN

LAN: Local Area Network ‚Äî small area like home or office.
MAN: Metropolitan Area Network ‚Äî covers cities or campuses.
WAN: Wide Area Network ‚Äî connects multiple LANs across countries.
